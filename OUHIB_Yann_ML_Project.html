Expliquons comment fonctionne les modles :

  • (1) corespond en fait un modle avec 1 seule variable explicative retenue (la variable retenue est celle dont la colonne est rempli par un 'asterisque')

  • etc...

On constate que les modles forward, backward et stepwise ont tous retenu les mmes variables expliques par modle.

Conclusion¶

On constate que le test d'autocorrlation est compris entre 0 et 2 ce qui est habituel pour une time serie.
Concernant les features que sont le type de proprit, code postale, date et nombre de chambres : AIC nous permet de savoir quelle feature a le moins d'impacte lorsqu'elle est retire du modle. Il s'agit ici du type de proprit, ce qui semble cohrent tant donne qu'il s'agit juste d'un 0 ou 1 dans notre dataset et qui a pour unique information de qualifi de "maison" ou "d'appartement" le bien. Le feature avec le plus de poids tant le nombre de chambres, ce qui encore une fois est cohrent avec la ralit et ce que recherche la plupart des gens lors d'un achat de proprit.
Enfin, on constate que le coefficient de dtermination est ~ 0.5 ce qui nous permet avec ce modle d'expliquer 50% de la variance des donnes prdictibles.
